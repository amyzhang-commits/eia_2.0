# Thermoelectric Datasets

This folder contains EIA thermoelectric cooling system data.

## Data Source

Download datasets from the [EIA Form 860 Survey](https://www.eia.gov/electricity/data/eia860/)

## Expected Files

Place the following files in this directory:
- `cooling_detail_2014.xlsx`
- `cooling_detail_2015.xlsx`
- `cooling_detail_2016.xlsx`
- `cooling_detail_2017.xlsx`
- `cooling_detail_2018.xlsx`
- `cooling_detail_2019.xlsx`
- `cooling_detail_2020.xlsx`
- `Cooling_Boiler_Generator_Data_Detail_2021.xlsx`
- `Cooling_Boiler_Generator_Data_Detail_2022.xlsx`
- `Cooling_Boiler_Generator_Data_Detail_2023.xlsx`
- `Cooling_Boiler_Generator_Data_Detail_2024.xlsx`

## Note

Dataset files are excluded from version control due to their size. Download them locally to run the analysis notebooks.
